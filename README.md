# lamp-autoinstaller
this script automatically installs lamp server

## How to run
Download the file

``
sudo wget https://raw.githubusercontent.com/mn-Devs/lamp-autoinstaller/main/install.sh
``

give the script read and write acces

``
sudo chmod +x install.sh
``

run the script

``
bash install.sh
``

done!
